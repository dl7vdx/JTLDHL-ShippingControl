📋 Windows Task Scheduler – Polling automatisieren

Damit das Tool automatisch alle 2 Stunden die DHL-Status abruft, erstelle eine geplante Aufgabe in Windows. Öffne dazu eine Kommandozeile als Administrator und führe folgenden Befehl aus: 

schtasks /create /tn "JTL-ShippingTracker-Poll" /tr "python C:\jtl-tracker\app\scheduler.py" /sc hourly /mo 2 /f

Pfad anpassen falls das Tool woanders installiert ist.
